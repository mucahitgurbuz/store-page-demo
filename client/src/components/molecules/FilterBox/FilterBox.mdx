# FilterBox

- A generic filter box implemented to be used as a filter list with **checkboxes**.
- It has a **search functionalty** which you can input a search text and filter categories filtered accordingly.
- When isBusy props is true, loading bars are showed under search bar.

## Props

<Props of={FilterBox} />

```js
interface IFilterCategories {
  label: string
  count?: number
  slug?: string
  isSelected: boolean
}
```
