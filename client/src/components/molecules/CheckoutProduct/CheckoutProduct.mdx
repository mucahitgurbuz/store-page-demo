# CheckoutProduct

- This is a checkout list item which is designed to be displayed in chekout box.
- It displays **name**, **price** and **count** of the product which is in the bucket.
- With the **remove** and **add** button, related actions are dispatched and item count in the bucked can be manipulated.

## Props

<Props of={CheckoutProduct} />

interface IProduct {
  name: string
  price: number
  count: number
  itemSlug: string
}
