# ProductItem

This is a box which gives information about a single product and have a action button where it can be added or removed from the bucket.

- It displays **name**, **price** and **image** of the product.
- It gets the `isInBucket` state from the redux-store and displays **Add** if there is no same item in the bucket or **Remove** if there is a same item in the bucket.
- Clicking the action button related actions are dispatched.
