# ProductListCategoryToggle

- It is component with a toggle switch behaviour.
- It renders a button for each `ItemCategory` which is initialy **mug** and **shirt**.
- Selected itemCategory displayed differently and by clicking the unselected ones a set action dispatched to change current itemCategory.
